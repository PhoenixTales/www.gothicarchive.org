--- layout: raw --- Gothic - The Game - Systemanforderungen
 
Systemvoraussetzungen
Computer: Gothic benötigt einen PC mit mindestens 400 MHz. Ihr Computer muss über wenigstens 128 MB RAM und eine Grafikkarte mit mindestens 16 MB Speicher verfügen.
 
Betriebssystem: Sie müssen Windows 95/98 oder Windows 2000 installiert haben, um Gothic auf ihrem System spielen zu können.
 
Steuerung: Es wird eine Tastatur benötigt.
 
Laufwerke: Es werden ein 4fach-CD-ROM und 700 MB freier Festplattenspeicher vorausgesetzt.
 
Grafik: Gothic benötigt einen SVGA-Monitor und eine DirectDraw® kompatible Grafikkarte. Sie müssen DirectX®7 oder höher auf ihrem System installiert haben, um das Spiel benutzen zu können. (Auf der Gothic CD wird DirectX 7.0a mitgeliefert.) Gothic unterstützt auch erweiterte Grafikfeatures, wenn sie eine GeForce Grafikkarte benutzen.
 
Sound: Gothic funktioniert mit jeder Soundkarte, die mit DirectX 7 oder höher kompatibel ist.
